# Mira UI

This is the React frontend for Mira. The initial UI is based on the prototype in `refer/mock.jsx`.

## Stack

- Vite
- React
- Tailwind CSS
- lucide-react

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The current UI is a migrated version of `refer/mock.jsx`.
- No backend integration is included yet.
